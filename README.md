## About This Project

This repository contains reupload of a sentiment analysis project using **Dynamic Expert Consensus Ensembling**, originally developed as a group project at **ETH Zürich** during the Computational Intelligence Lab
The original work was collaborative.

### Academic Context
This project was submitted as coursework at ETH Zürich. It is shared here for demonstration purposes only and is not intended as a reference solution for current students of the course.

## FeelingFinders
# CIL Project

`train_and_save_model.py` trains and saves our model results,  
while `dece_and_baselines.ipynb` uses these models in an ensemble to produce the final results and the plots that can be found in the report.

The required data is provided in the `data/` folder.  
All preprocessing techniques can be found in the `preprocessing/` folder.
